# 115RenamePlus

115 网盘视频文件自动改名工具，根据文件名中的番号自动查询并修改为标准格式。

修改自 [115RenamePlus](https://github.com/LSD08KM/115RenamePlus)

---

## 📦 安装

### 前提条件

1. 安装浏览器扩展：**[Tampermonkey](https://www.tampermonkey.net/)**
2. 确保浏览器能正常访问以下网站：
   - JavBus: `https://www.javbus.com/`
   - JavDB: `https://javdb.com/`
   - FC2: `https://adult.contents.fc2.com/`

### 安装步骤

1. 点击 [安装脚本](https://raw.githubusercontent.com/Oissp/115RenamePlus/master/115RenamePlus.user.js)
2. Tampermonkey 会自动打开并提示安装
3. 点击"安装"完成

### 更新

Tampermonkey 会自动检查更新，也可以手动：
- 打开 Tampermonkey 仪表盘
- 找到 `115RenamePlus`
- 点击"检查更新"

## 🎯 功能特性

### 支持的改名方式

| 改名方式 | 说明 | 适用场景 |
|---------|------|---------|
| **JavBus 改名** | 通过 JavBus 查询番号信息 | 大多数日系影片 |
| **JavDB 改名** | 通过 JavDB 查询番号信息 | 推荐，数据更准确 |
| **FC2 改名** | 通过 FC2 官网查询 | FC2-PPV 系列 |

### 支持的番号格式

- **标准番号**: `ABC-123`、`STARS-590`
- **FC2 番号**: `FC2-PPV-123456`、`FC2PPV-123456`
- **无码番号**: `CARIB-123456`、`1PONDO-123456`
- **特殊番号**: `T28-123`、`N-1234`

### 智能识别

- ✅ **多分段识别**: `CD1`、`HD2`、`FHD3`、`-A`、`_B`、`-3`、`_4`
- ✅ **中文字幕识别**: `-C`、`中字`、`中文字幕`
- ✅ **4K 识别**: `-4K`、`H264 版 `、`VP9 版`
- ✅ **引流站前缀清理**: `489155.com@`、`hhd800.com@`、`4k2.me@` 等

## 📝 改名格式

### 标准格式

```
番号_分段-C-4k 演员 标题 发行日
```

### 示例

| 原文件名 | 改名后 |
|---------|--------|
| `ABC-123.mp4` | `ABC-123 演员 A 标题 2024-01-15.mp4` |
| `ABC-123-C.mp4` | `ABC-123-C 演员 A 标题 2024-01-15.mp4` |
| `ABC-123-4K.mp4` | `ABC-123-4k 演员 A 标题 2024-01-15.mp4` |
| `ABC-123-CD2.mp4` | `ABC-123_2 演员 A 标题 2024-01-15.mp4` |
| `STARS-590A.mp4` | `STARS-590_A 演员 A 标题 2024-01-15.mp4` |
| `FC2PPV-123456.mp4` | `FC2-PPV-123456 演员 A 标题 2024-01-15.mp4` |
| `489155.com@FC2PPV-123456-2.mp4` | `FC2-PPV-123456_2 演员 A 标题 2024-01-15.mp4` |

### 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| **番号** | 标准番号格式 | `ABC-123`、`FC2-PPV-123456` |
| **_分段** | 多集时分段标识（可选） | `_1`、`_2`、`_A`、`_B` |
| **-C** | 中文字幕标识（可选） | `-C` |
| **-4k** | 4K 画质标识（可选） | `-4k`、`-4kH264 版` |
| **演员** | 女演员名单（JavDB 仅女演员） | `演员 A,演员 B` |
| **标题** | 影片标题 | `标题` |
| **发行日** | 发行日期 | `2024-01-15` |

## 🚀 使用方法

### 基本步骤

1. 打开 [115 网盘](https://115.com/)
2. 进入视频所在文件夹
3. **选中**需要改名的视频文件（支持多选）
4. **右键**点击选中的文件
5. 选择改名方式：
   - `视频改名 javbus` - 通过 JavBus 查询
   - `视频改名 javdb` - 通过 JavDB 查询（推荐）
   - `视频改名 FC2` - 通过 FC2 官网查询

### 使用技巧

| 场景 | 推荐方式 | 说明 |
|------|---------|------|
| 普通番号 | JavDB | 数据准确，演员信息完整 |
| FC2 番号 | FC2 改名 | 直接查询 FC2 官网 |
| 无码番号 | JavBus | 无码资源覆盖更全 |
| 批量改名 | 任意方式 | 支持多选批量处理 |

### 注意事项

1. **文件名必须包含番号**，且番号格式正确
2. **一次不要选择太多文件**，建议 10 个以内
3. **确保网络畅通**，需要访问查询网站
4. **改名后无法撤销**，建议先测试单个文件

## 🔧 高级功能

### 引流站前缀清理

自动清理常见引流站前缀，避免影响番号识别：

```
489155.com@
hhd800.com@
4k2.me@
fun2048.com@
big2048.com
...
```

### 分段格式统一

自动识别并统一多种分段格式：

| 原格式 | 统一后 |
|--------|--------|
| `CD1`、`CD2` | `_1`、`_2` |
| `DISC1`、`DISC2` | `_1`、`_2` |
| `PART1`、`PART2` | `_1`、`_2` |
| `-A`、`-B` | `_A`、`_B` |
| `-1`、`-2` | `_1`、`_2` |

### 中文字幕识别

智能识别中文字幕标识，避免误判：

- ✅ 正确识别：`-C`、`中字 `、` 中文字幕`
- ❌ 避免误判：`.com` 中的 `c`、番号中的 `C`

## ❓ 常见问题

### Q: 右键菜单不显示？

**A:** 检查以下几点：
1. 确认已安装 Tampermonkey 扩展
2. 确认脚本已启用（Tampermonkey 仪表盘查看）
3. 刷新网页（Ctrl+F5 强制刷新）
4. 检查浏览器 Console 是否有报错

### Q: 改名失败？

**A:** 可能原因：
1. 文件名中不包含有效番号
2. 番号格式特殊，无法识别
3. 查询网站无法访问
4. 网络问题导致查询超时

### Q: 演员名包含男演员？

**A:** JavDB 改名默认只提取女演员。如果仍有男演员，可能是：
1. JavDB 页面结构变化
2. 该影片演员信息特殊
3. 建议改用 JavBus 改名

### Q: 分段识别错误？

**A:** 检查文件名格式：
1. 避免文件名中包含日期（如 `2024-01-15`）
2. 避免特殊字符干扰
3. 可以在 GitHub 提 issue 反馈
